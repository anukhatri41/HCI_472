Navigation Bar

The responsive navigation bar allows users to navigate the site and changes between links and the hamburger icon depending on the screen size.

Example

  • Portfolio
  • About Me
  • LinkedIn
  • Resume

Code

                  
    <nav class="navbar navbar-expand-lg nav-custom">
      <div class="container-fluid">
        <a class="navbar-brand" href="#">
          <img src="../images/Primary3.svg" alt="Primary Logo - Anu Khatri" height="50px">
        </a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
          <img src="../images/Icon-Menu.svg">
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
          <ul class="navbar-nav ms-auto">
            <li class="nav-item">
              <a class="nav-link active nav-underline nav-text" aria-current="page" href="#">Portfolio</a>
            </li>
            <li class="nav-item">
              <a class="nav-link nav-text" href="#">About Me</a>
            </li>
            <li class="nav-item">
              <a class="nav-link nav-text" href="#">LinkedIn</a>
            </li>
            <li class="nav-item">
              <a class="nav-link nav-text" href="#">Resume</a>
            </li>
          </ul>
        </div>
      </div>
    </nav>

Usage

  • The navigation bar should be used on all pages
  • The navigation bar should be consistent on all pages
  • The navigation bar should be sticky to the top
  • The logo in the navigation bar should link to the home page

Accessibility Considerations

  • The navigation has a high color contrast, meeting the WCAG color requirements
  • The navigation bar is consistent with other sites for the desktop and mobile version with recognizable icons
  • The navigation bar is consistent across all pages allowing for better learnability
  • The navigation bar has large targets for page links
  • The navigation bar uses underlines to indicate the active tab instead of relying on color
