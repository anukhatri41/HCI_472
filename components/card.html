Card

The card gives a preview for a project and allows the user to navigate to the case study.

Example

Title

CATEGORIES

View Case Study

Code

                  
    <div class="col-lg-4 col-md-6 py-2">
      <div class="card h-100 bg-card card-custom">
        <div class="image-wrapper">
          <img class="card-image" src="../images/Image_Placeholder.png" alt="Image Placeholder">
        </div>
        <div class="card-body body-custom">
          <div class="card-titles">
            <p class="card-title large cards">Title</p>
            <p class="subtitle cards">
              CATEGORIES
            </p>
          </div>
          <a href="#" class="btn btn-custom w-100" role="button">View Case Study</a>
        </div>
      </div>
    </div>

Usage

  • The card should have the project title and the related categories, such as UX Design or UX Research
  • The card image should be relevant to the project

Accessibility Considerations

  • The card text and background have high contrast
  • The button text and background have high contrast in normal and hover state
  • The image should have appropriate alt text
