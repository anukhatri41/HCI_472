Footer

The footer reiterates links found in the navigation bar and includes copyright information.

Example

Portfolio About Me LinkedIn Resume
anukhatri41@gmail.com

Â© Anu Khatri

Code

                  
    <footer class="text-center text-lg-start footer-frame">
      <div class="footer-elem">
        <div class="logo-footer">
          <img src="../images/Secondary_1.svg" width="155px">
        </div>
        <div class="footer-text">
          <div class="footer-links">
            <a href="#" class="footer">Portfolio</a>
            <a href="#" class="footer">About Me</a>
            <a href="#" class="footer">LinkedIn</a>
            <a href="#" class="footer">Resume</a>
          </div>
          <div>
            <a href="#" class="footer">anukhatri41@gmail.com</a>
          </div>
          <div>
            <p class="subtitle footer">© Anu Khatri</p>
          </div>
        </div>
      </div>
    </footer>

Usage

  • The footer should be at the bottom of every page
  • The footer should include the secondary logo, navigation links, email address, and the copyright information

Accessibility Considerations

  • The navigation has a high color contrast, meeting the WCAG color requirements
  • The footer is consistent across all pages allowing for better learnability
  • The footer has large targets for page links
  • The footer uses underlines to indicate navigable links
  • The footer uses the footer HTML tag and contains information normally found in a footer
